- 공동 강의/교재 가능하나 타겟층 차이로 시너지 제한적

## 4. Mnemo 비상회의 시사점 — 3인 종합

### 공통 성공 패턴 (3인 모두 해당)
1. **일관된 콘텐츠 생산** — 주 1회 이상, 2년+ 지속
2. **무료 → 유료 퍼널** — 무료 콘텐츠로 신뢰 → 유료 전환
3. **다층 수익원** — 단일 수입 의존 X (강의+출판+컨설팅+구독)
4. **도구가 아닌 철학** — 단순 기술이 아닌 "사고 방식" 판매
5. **개인 브랜드** — 조직이 아닌 개인의 전문성이 핵심 자산

### 핵심 차별화 — Mnemo만의 고유 포지션

```
ShaeO:  인문학 + AI 비판적 사용 + 글로벌 오디언스
김우정:  AI 스토리텔링 + 멀티미디어 실전 + LinkedIn
홍순성:  AI 글쓰기 도구 + 출판 다작 + 매일 콘텐츠

Mnemo:  인지과학 + 경영전략 + 논리적 글쓰기 + AI 에이전트 자동화
        ↓
  "이론적 깊이 + 시스템적 자동화 + 다국어 확장"
  = 3인 누구도 점유하지 않는 고유 영역
```

**Mnemo의 미점유 니치**: "AI 시대의 전략적 사고 프레임워크를 에이전트로 자동화하는 인문학자"
- ShaeO는 AI를 제한적으로 사용 → Mnemo는 적극 활용
- 김우정은 실전 도구 중심 → Mnemo는 이론+시스템
- 홍순성은 B2C How-to → Mnemo는 B2B 전략 컨설팅
