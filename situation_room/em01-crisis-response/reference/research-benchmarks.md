## B. 1인 크리에이터 / 솔로프리너 모델 (핵심 참고)

### 5. 조쉬 — AI 솔로프리너 클럽 (ASC) ⭐ 최우선 벤치마크
- **프로필**: 前 회사원 → 퇴사 후 1인 기업가. 뉴스레터 5만+ 구독자, 2,000+ 고객 판매, 전 회사 연봉 2배 수익
- **사업 구조**:
  - **콘텐츠**: 뉴스레터 (Claude 활용 주간 발행) → 오디언스 구축
  - **커뮤니티**: ASC (AI 솔로프리너 클럽) — 연간 멤버십 월 12만원 (144만원/년)
  - **교육**: 3트랙 (크리에이터/빌더/세일즈) — 4주 집중 과정
  - **강의**: 패스트캠퍼스 바이브코딩 강의
  - **이벤트**: 트레바리 "1인 기업의 모든 것", 솔로프리너 컨퍼런스
  - **SaaS**: 바이브코딩으로 직접 제작 (DividendPro 등)
- **핵심 전략**:
  1. 무료 콘텐츠(뉴스레터, SNS) → 오디언스 구축
  2. 커뮤니티 멤버십 → 안정 수익
  3. 강의·이벤트 → 브랜드 확장
  4. SaaS → 스케일러블 수익
- **Mnemo 적용 가능 포인트**:
  - GnoTech 지식 시스템을 콘텐츠화 → 뉴스레터/블로그로 오디언스 구축
  - EIU 진단 도구를 무료 버전으로 리드 생성
  - 자소서 관련 무료 콘텐츠 → 유료 서비스 전환 퍼널
  - 커뮤니티 모델은 중기 목표로 설정

## E. 핵심 시사점 종합

### Mnemo의 차별화 자산
1. **전략 컨설턴트 + 소설작가** — 논리적 구조화 + 감성적 서사의 양면
2. **AI 에이전트 시스템 직접 구축** — GnoTech, EIU-resume GPTs, 멀티에이전트 거버넌스
3. **독자적 프레임워크** — EIU(수용성 진단), CB-Matrix(메타인지), LSAi(교육 통합)
4. **소설 창작물** — 와디즈 3무서점 선정

### 현재 가장 부족한 것 (BM 대비)
1. **SNS 오디언스 = 0** — 조쉬는 5만+, 정선비/김지나도 활발. Mnemo는 채널 자체가 없음
2. **무료 콘텐츠 부재** — 무료 진단, 무료 자료, 블로그 글 등 리드 생성 장치 없음
3. **린크트리/포트폴리오 페이지 없음** — 눈오지, 대기업 김과장 모두 litt.ly 활용
4. **숫자로 증명 부재** — "연 1,000건", "500곳 합격" 같은 구체적 성과 수치 없음

### 즉시 실행 가능한 BM 적용
1. **litt.ly 또는 린크트리 생성** — 프로필 + GPTs 링크 + 서비스 소개 (비용: 0원)
2. **네이버 블로그 또는 브런치 개설** — 자소서 전략 글 주 2~3회 (비용: 0원)
3. **EIU-resume GPTs를 무료 진단 리드 도구로 전면 배치** — 견적서에 GPTs 링크 포함
4. **견적서 재구성** — 무료 진단 → 저가 자료(3~5만원) → 풀 서비스(10만원+) 3단계 퍼널
