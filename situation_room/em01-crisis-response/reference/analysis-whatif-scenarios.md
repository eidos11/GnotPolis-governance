# §2.5 What-If 시나리오 분석

> 작성: 2026-03-18 | 6주 집중(~4/28) + Decision Gate 기준
> 원칙: 간결·명확·실무적. 유연한 변경 적용 가능한 구조.

---

## 구조: 3개 시나리오 × 2개 페이즈

```
         3월 하반기 ~ 4월 (6주 집중)
                    │
              ══════╪══════
             /      │      \
        시나리오A  시나리오B  시나리오C
        (순풍)    (기본)    (역풍)
                    │
            ════ Decision Gate ════
              4월 마지막 주 판단
                    │
              ╔═════╧═════╗
              ║ 계속 vs 전환 ║
              ╚═══════════╝
```

---

## 시나리오 A: 순풍 — "견인력 확보"

### 조건 (이 중 2개 이상 충족)
- 와디즈 펀딩 목표 50%+ 달성
- Primer 미팅 성사 또는 긍정 피드백
- LinkedIn 팔로워 100+ 또는 의미 있는 B2B 접촉 발생
- 롤모델(ShaeO/김우정) 중 1인 이상 협력 응답

### 6주간 체감
- SNS 포스팅에 반응이 오고, DM이나 문의가 간헐적으로 발생
- 와디즈 후원자가 실제로 생기면서 "시장이 존재한다"는 증거 확보
- Primer에서 "한번 만나보자"는 연락이 오거나, 지원서가 다음 단계로 진행

### Decision Gate 판단
**→ 계속 집중 (5~6월)**
- 수익원이 가시화되었으므로 전력 투자 지속
- 구독 비용 현행 유지, 병렬 세션 강화
- 취업 활동은 최소화하되 좋은 기회는 열어둠

### 5~6월 방향
- 와디즈 펀딩 완주 + 2차 프로젝트 기획
- Primer 후속 미팅/투자 협의
- LinkedIn 콘텐츠 누적 → 강의/컨설팅 문의 대응
- 6월 말까지 월 100만원+ 수익 경로 확보 목표

---

## 시나리오 B: 기본 — "씨앗은 뿌렸으나 아직 싹이 안 보임"

### 조건 (가장 높은 확률)
- 와디즈 등록은 했으나 펀딩 10~30% 수준 (목표 미달이지만 0은 아님)
- Primer 지원서 제출했으나 응답 대기 중
- LinkedIn 포스팅 시작했으나 팔로워 30~50명, 반응 미미
- 롤모델 접촉 시도했으나 아직 회신 없음
- 무료 자료 배포했으나 다운로드 수십 건 수준

### 6주간 체감
- "뭔가 움직이기는 하는데 아직 확실한 신호가 없다"
- 포스팅은 하고 있지만 바이럴은 안 됨
- 돈은 안 들어오지만, 기반은 쌓이고 있다는 느낌

### Decision Gate 판단
**→ 이중 트랙 (5월)**
- 취업 활동 **본격 개시** (서류 지원 주 5건+, 면접 응시)
- 동시에 SNS·와디즈·Primer **유지** (포기가 아닌 병행)
- AI 구독 축소 검토 시작 (Gemini/Perplexity 해지, GPT Plus 전환)

### 5~6월 방향
```
시간 배분 전환:
  기존: 프로젝트 100%
  전환: 취업 60% + 프로젝트 40% (유지·관리 중심)

유지할 것:
  - LinkedIn 주 1~2회 포스팅 (축소하되 중단하지 않음)
  - 와디즈 펀딩 진행 중이면 관리
  - Primer 응답 오면 대응

축소할 것:
  - 새로운 콘텐츠 대량 생산 → 기존 자료 재활용 위주
  - 에이전트 팀 활용 → 최소 필요 시에만
  - AI 구독 → 최소 스택 (Claude Max + GPT Plus + Notion)
```

### 분기 2차 판단 (5월 중순)
- 취업이 먼저 성사되면 → 프로젝트는 여가 시간 병행
- Primer 미팅 성사 또는 와디즈 급성장 시 → 다시 시나리오 A로 전환
- 둘 다 안 되면 → 시나리오 C 방향

---

## 시나리오 C: 역풍 — "신호 없음"

### 조건
- 와디즈 펀딩 실패 또는 등록 자체 지연
- Primer 무응답 또는 탈락
- LinkedIn 포스팅했으나 반응 거의 0
- 롤모델 접촉 전부 무응답
- 무료 자료 배포해도 다운로드 한 자릿수

### 6주간 체감
- "열심히 했는데 시장이 반응하지 않는다"
- 자금 고갈이 현실적 압박으로 다가옴
- 구독 비용 자체가 부담

### Decision Gate 판단
**→ 생존 모드 전환 (즉시)**
- 취업 활동 **전력** (프로젝트는 최소 유지)
- AI 구독 **Claude Max만 유지**, 나머지 전부 해지
- 프로젝트는 "동결이 아닌 최소 유지" — 주 3~5시간만 투입

### 5~6월 방향
```
취업 전력:
  - 강사, 경영컨설턴트, 시설물관리자 — 기존 경험 분야 집중
  - 프로젝트 병행 가능한 직종 우선
  - 목표: 5월 내 채용 확정

프로젝트 최소 유지:
  - GitHub repo 관리 (주 1회)
  - 노션 지식베이스 유지 (수정 없이 보존)
  - LinkedIn 월 1~2회 포스팅 (완전 중단 방지)
  - 취업 성공 후 여가 시간에 재개 계획 수립

보존할 자산:
  - 노션 5영역 전체 (Business 요금제는 유지 검토)
  - GnoTech Git 워크스페이스 (비용 0)
  - 완성된 콘텐츠 (Howstudy, RG, 소설 등)
  → 이 자산은 사라지지 않으므로 취업 후 재개 가능
```

## Decision Gate 판단 기준 (4월 마지막 주)

단순하게 **3개 신호**만 확인:

| 신호 | 측정 | A (순풍) | B (기본) | C (역풍) |
|------|------|---------|---------|---------|
| **돈이 들어왔는가** | 와디즈 후원금, 서비스 매출, 기타 수익 | 50만원+ | 1원~50만원 | 0원 |
| **사람이 반응했는가** | Primer 응답, 롤모델 회신, B2B 문의 | 2건+ 의미있는 접촉 | 1건 또는 대기 중 | 0건 |
| **기반이 쌓였는가** | LinkedIn 팔로워, 콘텐츠 반응, 다운로드 | 팔로워 100+ | 30~100 | 30 미만 |

**판단 로직:**
- 3개 중 **2개 이상 A** → 시나리오 A
- **B가 다수** 또는 **A+C 혼재** → 시나리오 B
- 3개 중 **2개 이상 C** → 시나리오 C

---

## 핵심 원칙

1. **시나리오는 고정이 아니라 신호에 따라 전환** — 3주차에 갑자기 순풍이 불면 A로, 역풍이 오면 C로 즉시 전환
2. **복잡한 스케줄보다 "오늘 가장 중요한 1가지"에 집중** — 매일 아침 "오늘의 최우선 액션 1개" 결정
3. **모든 시나리오에서 프로젝트 자산은 보존** — 최악의 경우에도 지식·콘텐츠·시스템은 남아 있으므로 재개 가능
4. **취업은 패배가 아니라 전략적 리셋** — 안정적 수입 확보 후 프로젝트 재개가 오히려 더 건강한 경로일 수 있음
