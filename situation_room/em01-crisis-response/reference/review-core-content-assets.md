### 1.4 CB-Matrix와의 관계

Howstudy 1.0은 CB-Matrix의 **A사분면(확실성 영역)에 해당하는 방법론**:
- 불확실성의 극단적 상황에서도 확실한 영역과 방식이 가능한 부분은 구분하여 처리
- 이 구분 자체가 CB-Matrix의 기본 방식이자 AI 시대에 유효한 핵심 방법론
- "AI가 대신 공부해줄 수 없는 영역"이라는 메시지가 콘텐츠로서 강력

## 2. RG 교재 — 두 작품의 본질적 구분

### 2.1 핵심: 문학적 vs 논증설명적 생성

Mnemo가 도출한 AI 저술의 2대 축 구분 (별도 방법론 정리 진행 중):

| 구분 | 사유의 도시 (시간탐정단) | 3부작 (라운드테이블) |
|------|----------------------|-------------------|
| **장르** | 문학적 — 소설/드라마/서사 | 논증설명적 — 에세이/대화형 교양서 |
| **Mnemo 역할** | **구상자** — 배경·주제·인물·스토리라인 설계 (소설가 역할) | **아이디어 제시자** — 간략 방향만 제시 |
| **AI 역할** | 구체적 스토리·상황·대사를 독자적 서술 (GPT) | 기획+구성+서술 전과정 자율 수행 (Perplexity 멀티모델) |
| **기반 자료** | Mnemo의 직접 구상 (원본은 GPT 채팅에서 생성) | VS 기획서를 AI가 토대로 활용 |
| **정련 과정** | 원본 → 가이드 형식 재작성 (서식 통일, 출판 수준 완성도) | AI 자율 구성 (추가 정련 최소) |
| **품질 향상 방향** | 서식·구성 정련 (이미 달성) | RAG(원저작 DB) + GraphRAG(관계성) 보강 |

### 2.2 작품 A: "사유의 도시: 시간탐정단" (문학적 생성)

**로컬 위치**: `P11_VirtualSymposium/ReGenesis/docs/RG_pilot-sayuicity-ch1-2_v0.2.md`

| 항목 | 내용 |
|------|------|
| **장르** | 학습 판타지 소설 (YA + 교육) |
| **타겟** | 중·고등학생 (번역/글로벌 확장 전제) |
| **주인공** | 정우 — 고등학교 2학년, 딥페이크 사건 계기로 "사유의 도시" 진입 |
| **구성** | 현실(SNS/딥페이크) + 사유의 도시(철학자 만남) 교차 |
| **Ch1** | "동굴 밖의 빛은 눈부시다" — 소크라테스/플라톤, 진리란 무엇인가 |
| **Ch2** | "예성의 무너진 성벽" — 공자/맹자/홉스/루소/노자, 질서란 무엇인가 |
| **특수 장치** | Fact Room — 본문의 드라마/추리 후에 정확성/교육 가치를 후처리 보강 |
| **영문판** | `RG_pilot-sayuicity-ch1-2-en_v0.2.md` 존재 |

**생성 방법 상세 (Mnemo 직접 설명):**
1. Mnemo가 GPT 채팅에서 배경·세션주제·주요인물·핵심 스토리라인을 마치 소설가가 플롯 구성하듯 제시
2. GPT가 이에 기반하되 구체적 스토리와 상황은 독자적으로 서술
3. Mnemo는 직접적 서술에 전혀 개입하지 않고 구상만 제시
4. 이후 새로운 가이드의 정제된 형식·서식으로 재작성 — 내용 대부분 유지, 서식 통일·정련되어 출판 가능 수준으로 완성도 상승

**품질 평가:**
- 서두(딥페이크→교실→"진리는 무엇인가?" 시험 문제→사유의 도시 진입)의 전환이 자연스럽고 몰입도 높음
- 현대 10대의 현실(SNS, 확신 vs 진리)과 고전 철학의 교차가 상업적 경쟁력 보유
- Fact Room 장치로 "재미 → 깨달음" 루프가 구조적으로 설계됨

### 2.3 작품 B: "3부작 라운드테이블" (논증설명적 생성)

**로컬 위치**: `P11_VirtualSymposium/ReGenesis/relevant/RG_relevant-sample-draft_v1.md`

**생성 방법 상세 (Mnemo 직접 설명):**
1. Mnemo가 간략한 아이디어만 제시
2. Perplexity에서 GPT와 여러 모델들이 합작하여 기획·구성·서술 전과정 수행

- 전문적 지식과 대가들 원저작 DB를 **RAG 형태로 연결**

- 이를 통해 학술적 정확성과 논증 깊이 대폭 향상 가능

## 4. 추가 고려사항 통합

### 4.1 AI 구독 비용 정정 (3/18 확정)

GPT Pro는 이벤트로 Plus 가격(~$20)에 2개월간(~5/10) 이용 가능.
현재 전체 AI 도구 비용은 **거의 필요 최저 수준**이며, 이벤트 종료 후에도 Claude Max + GPT Plus + Notion = ~$130/월로 핵심 유지 가능.

### 4.2 GnoTech 구축 기간 정정

"4년 준비"는 오류. **1년간 AI 기술 학습과 동시에 GnoTech + GnotPolis 전체 시스템 구축 완료.** 이 집중 구축력 자체가 강점.

### 4.3 동원과기대 관련 정정

"강연 경험"은 에이전트 오류. 정확한 사실:
- 동원과기대 **전략 컨설팅 프로젝트** — 가장 성공적인 프로젝트
- 극도로 어려운 조건: 원래 1~2개월 필요한 정부 사업계획서를 **2주 만에 완료**
- 실제 정부 과제 선정 → 사업기간 전체 **총 200억원 정도 지원** 확보
- 대학측에 결정적 위기극복과 발전발판 마련
