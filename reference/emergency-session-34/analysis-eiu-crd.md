# EIU_CRD 프레임워크 분석: 므네모 서비스 현황 진단

## 개요

이 문서는 므네모의 8개 서비스/채널을 **EIU_CRD 프레임워크**로 종합 평가하고, 숨고 서비스 실패(27건 견적 → 0 응답)의 근본 원인을 진단하며, 향후 우선순위를 제시합니다.

**핵심 발견**: 숨고 서비스의 핵심 문제는 **I(Interest) 차원의 완전한 실패**이며, E와 U도 동시에 약화되어 있는 상황입니다. 반면 와디즈·LinkedIn·Primer 등의 채널은 훨씬 높은 I와 U를 확보할 수 있습니다.

---

## 1단계: EIU_CRD 프레임워크 정의

### 1.1 EIU 3요인 정의 (완성도: 90%)

EIU 프레임워크는 어떤 **서비스 또는 채널**이 시장에서 채택될 가능성을 3가지 정량 지표로 평가합니다.

| 요인 | 약자 | 정의 | 평가 기준 | 측정 단위 |
|------|------|------|----------|---------|
| **Easiness** | E | 고객이 서비스에 **접근하고 이해하기 쉬운 정도** | 진입장벽 낮음, 설명 명확, 신뢰도 입증 | 1~5점 |
| **Interest** | I | 고객이 서비스에 **관심을 갖게 되는 정도** | 문제 인식, 차별화 명확, 설명의 임팩트 | 1~5점 |
| **Utility** | U | 고객이 서비스로부터 얻을 **실질적 이득이 명확한 정도** | 구체적 결과, 비용 대비 가치, 실행 용이성 | 1~5점 |

### 1.2 게이트-엔진 구조 (세그먼트별 가중치)

EIU 점수는 **고객 세그먼트별로 다른 가중치**를 받습니다.

```
세그먼트 분류 (Rogers/Moore 기반):
  
  1. 대중(Mass Market) — 비기술자, 빠른 의사결정
     가중치: E(40%) > U(40%) > I(20%)
     특징: 진입 용이성과 명확한 효과가 최우선
     
  2. 실용주의자(Pragmatists) — 직장인, 신뢰도 중시
     가중치: U(40%) > I(35%) > E(25%)
     특징: 효용성과 차별화 명확성이 중요
     
  3. 전문가(Experts) — 학자, 깊이 추구
     가중치: I(45%) > U(35%) > E(20%)
     특징: 흥미로운 아이디어와 이론적 가치 추구
```

### 1.3 최종 점수 산정

```
EIU 종합점수 = (E×가중치_E + I×가중치_I + U×가중치_U) / 100
범위: 1~5점
```

### 1.4 CRD 보조 평가 (신뢰도, 경합성, 실행성)

| 약자 | 정의 | 측정 내용 |
|------|------|---------|
| **C (Credibility)** | 신뢰도 | 고객 리뷰, 증명된 실적, 전문성 입증 정도 |
| **R (Reach)** | 도달성 | 채널의 기존 오디언스 규모 또는 확보 가능성 |
| **D (Differentiation)** | 차별화 | 경쟁사 대비 명확한 차별점 존재 여부 |

## 2단계: 숨고 서비스 실패 진단 (27견적 → 0응답)

### 2.1 근본 원인: 3차원 동시 실패

#### E(Easiness) 차원: 중등도 문제

**문제점**:
- 신뢰도 기반 부족: 리뷰 0개 (숨고 알고리즘 콜드 스타트)
- 경쟁자 기준 낮음: 상위권(노영우 750리뷰, 문근영 99K/시간) 대비 절대적 신뢰도 격차
- 신규자 성공률 60% (숨고) vs 80%(Lapeed) → 플랫폼 구조적 난제

**증거**:
- research-platforms.md: "Soomgo 신규 판매자 성공률 60%"
- research-competitors.md: "노영우 750 reviews(상위 0.01%), 문근영 99K/시간"

#### I(Interest) 차원: **치명적 실패** ⚠️

**문제점**:
- 서비스 설명 추상성: "상담형", "진단가이드형" → 고객이 **무엇을 받을지 불명확**
- 포지셔닝의 모호성: "전략컨설턴트 + 소설가" → 고객 입장에서 "왜 이 사람의 자소서 조언을 받아야 하는가?" 불명확
- 무료 콘텐츠/신호 없음: LinkedIn 프로필 없음, 블로그 없음, 쓰레드 포스트 없음 → 신뢰 신호 전무
- 차별화 불명확: "자소서 컨설팅"은 이미 시장에 포화된 카테고리 (상위권 다수)

**증거**:
- research-competitors.md: "우상단 경쟁자들 평균 경력수치화: 150회+ 컨설팅, 500명+ 합격사례"
- research-competitors.md: "성공 패턴: 대기업 경험 강조 → 신뢰성"
- 현황: 숨고 프로필에 **구체적 사례/수치 전무**

**고객 심리 분석**:
```
고객이 묻는 질문:
  Q1: "이 사람 자소서 잘 봐?" 
      A: 리뷰 0 → 알 수 없음 (E 실패)
  
  Q2: "이 사람이 나와 맞나?" 
      A: 설명 추상적 → 감 잡을 수 없음 (I 실패)
  
  Q3: "실제로 도움이 될까?" 
      A: 구체적 사례 없음 → 확신 불가 (U 실패)
```

#### U(Utility) 차원: 약한 설명

**문제점**:
- 구체적 결과 미제시: "자소서 개선" → 얼마나 개선? 어떻게 개선?
- 비용 대비 가치 불명확: 숨고 평균 30-50K → "그 돈을 내가 써도 될까?" 의심
- 실행 용이성 불명확: "1회 상담" vs "5회 지도" vs "최종 첨삭"? 명시 없음

**증거**:
- research-platforms.md: "Soomgo 평균 가격 40-80K (프리미엄 110K)"
- research-competitors.md: "성공자들의 무료→유료 퍼널: 4단계 (콘텐츠→신뢰→유료→재구매)"
- 현황: 므네모는 1단계(콘텐츠)조차 없음

### 2.2 비교 분석: 왜 상위권은 성공하는가?

**노영우 사례** (750리뷰, 5.0/5.0, 상위 0.01%):
- E: ✅ 750리뷰 = 신뢰도 극상
- I: ✅ "대기업 3사 인사담당" → 차별화 명확
- U: ✅ 구체적: "삼성·SK·현대 합격자 다수" + "이력서→자소서→면접까지 연계"

**문근영 사례** (99K/시간, 프리미엄 포지셔닝):
- E: ✅ 높은 가격대 자체가 신뢰 신호
- I: ✅ "경영전략 13년 경력" → 전문성 명확
- U: ✅ "1시간 상담" → 구체적 서비스 정의

**므네모 현황**:
- E: 🔴 리뷰 0, 신뢰 신호 전무
- I: 🔴 포지셔닝 불명확, 무료 콘텐츠 전무
- U: 🔴 구체적 서비스/결과 미제시

### 2.3 결론: 3가지 근본 원인

1. **신뢰도 기반 부족(E)**: 플랫폼 콜드 스타트 → **리뷰 5~7개 필요** (2월 견적 활동)
2. **포지셔닝 실패(I)**: "자소서 컨설팅"은 이미 포화 → **"AI 시대 전략적 글쓰기"로 상위 개념화 필요**
3. **무료 신호 전무(I+U)**: SNS·블로그 콘텐츠 없음 → **LinkedIn 포스트 + 무료 강연 시작**

### 3.2 EIU 평가 매트릭스

```
평가 기준:
  E (Easiness):  1=매우 높은 진입장벽, 5=완벽한 접근성
  I (Interest):  1=차별화 전무, 5=강한 관심 유발

    대중     : E(40%) + U(40%) + I(20%)
    실용주의자: U(40%) + I(35%) + E(25%)
```
