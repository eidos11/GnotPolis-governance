## 1. 자소서 서비스 재평가

### 선정 경위
- 원래 강점 분야: **학습법, 논리** (철학 전공, 경영컨설턴트 경험, 방대한 인문사회 독서)
- 숨고/크몽에서 학습법 대비 자소서 고객 기반이 **~10배** → 자소서로 결정
- 레드오션 인지했으나 **진입 자체의 고전은 예상 못함**

### 현재 판단
- 자소서 서비스는 **유지하되 전념 불필요**
- 서비스 자체는 완비 상태 — 고객 유입만 확보되면 경쟁력 충분
- **중기 방향**: 논리·전략 분야 중심 B2B 컨설팅
- **단기 목적**: 자소서 등 플랫폼 서비스로 운영자금 충당

### 시사점
- 자소서에 마케팅 비용/시간을 과다 투입하지 말 것
- Lapeed 등 저비용 플랫폼으로 전환 검토 (수수료 1.6%)
- 무료 진단(GPTs) → 자료 판매 → 서비스의 3단계 퍼널은 유지

## 2. 핵심 기대 방향 — 와디즈 + Primer

### 전략적 우선순위 (Mnemo 확정)

```
1순위: 와디즈 크라우드 펀딩 + Primer 인큐베이션
2순위: LinkedIn/Facebook 기반 전문가 네트워크 구축 + 홍보
3순위: 플랫폼 서비스 (자소서 유지, Lapeed 추가 등)
```

### 와디즈 전략

**3무서점 제약 사항**:
- ❌ 사업자등록 + 통신판매업 신고 필수 → **Mnemo 현재 해당 안됨**
- 홍보에서 전혀 언급 없었던 사항 (파악 지연)

**대안: 일반 펀딩 (사업자 불필요)**:
- **프로젝트 A**: 단편소설 3편 전자책 (원래 아마존 출판 예정이던 작품)
- **프로젝트 B**: RG프로젝트(P11) 논리·철학 대화편 → 학습교재/교양교재
  - 내용 상당히 완성된 상태
  - 흥미 유발 가능성 있음

**와디즈 펀딩의 본질** (Mnemo 분석):
- 한국 크라우드펀딩 ≈ 선주문 판매 시스템
- 구매자 = 투자자, 마케팅+판매루트를 사전 확보
- 모험적 투자보다는 **미리 계약 확보 후 제작·이행** 방식

### Primer 전략

**지원 방식**: 상시모집 vs 예비창업자 **둘 중 하나만** (동시 지원은 역효과)
- 심층 조사 후 결정
- **EIU-CRD 프로젝트** 중심으로 지원 검토
- 미팅 성사 시 GnoTech 전체 프로젝트 연관성도 어필
- 투자사 방향이 별도 프로젝트면 유연 대응 가능

## 3. 홍보 전략 — LinkedIn + Facebook 중심

### 채널별 현황

| 채널 | 상태 | 전략 |
|------|------|------|
| **LinkedIn** | 한글/영문 프로필 **완성** | 전문가 네트워크, B2B 접점, 프로젝트 소개 |
| **Facebook** | 20년 계정 보유 (알고리즘 유리), 최근 6개월 매일 사용 | 전문가/현업자 커뮤니티, AI 관련 네트워크 |
| **GitHub** | GnoTech Workspace + GnotPolis-governance 운영 중 | 프로젝트별 맞춤 repo, 홍보용 자료 정리 |

### Facebook 특수 상황
- 20년 전 계정 생성 → 장기 계정 알고리즘 유리할 수 있음
- 20년간 포스팅 거의 없었으나 최근 6개월 매일 1시간 정보확인용 사용 (Thread, Facebook)
- AI 발전 정보를 SNS에서 매일 선별 → 노션에 간이 큐레이션 형태로 축적 중
- 이 자료들이 콘텐츠 소재로 즉시 활용 가능

### 자동화 가능성
- 현재 GnoTech 프레임워크로 게시용 콘텐츠 **대량 생산 가능**
- Mnemo가 사전 검토 + 최종 확인에 개입 필수
- **일부 자동 포스팅도 가능한 상태**
- 3월 하반기에 가시적 기반 누적 → 펀딩 도전

### GitHub 활용 확장
- 프로젝트별 홍보용 맞춤 repo 생성 가능
- 깃허브에 자료 정리가 매우 신속·효율적으로 가능한 상태
- 투자자/파트너 대상 프로젝트 소개 자료 → GitHub Pages 배포

## 4. 노션 자산 (미활용 상태)

### 보유 자산
- **사업전략 + 중기 방향**: 매우 자세히 문서화된 상태
- **프로필**: 한글/영문 완성
- **사업계획서**: 구비
- **초기 아이디어~이론적/실무적 논의~기획**: 단계별 문서화
- **방대한 조사·자료 모음**: SNS에서 매일 선별 저장한 마케팅/AI 자료 (간이 큐레이션)

### 활용 방향
- Primer 지원 시: 노션 자료 → 1-pager/피치덱 변환
- LinkedIn/Facebook 포스팅: 노션 큐레이션 자료 → 콘텐츠 가공
- GitHub repo: 노션 프로젝트 자료 → 구조화된 공개 자료

## 5. 투자·고용·협업 가능성 (Mnemo 관점)

### 기대 시나리오 (복수)
1. **펀딩 성공**: 와디즈 전자책/교재 → 초기 매출 + 홍보 효과
2. **투자 유치**: Primer 등 → 6개월+ 운영자금 확보
3. **일감 수주**: 전문가 네트워크에서 B2B 컨설팅 기회
4. **AI 협업 파트너 고용**: 최근 트렌드 — 소규모 팀 + 각자 AI 팀 운영
5. **GnoTech 기반 고용 제안**: 프로젝트 공개 → 관심 있는 조직에서 제안

### 핵심 전제
- **폭넓고 충분한 전파**가 선행되어야 잠재 투자자/파트너 중 일부라도 관심
- 프로젝트 내용을 **실용화된 상태로** 제시해야 설득력
- 큰 투자가 아니어도 6개월 운영자금이면 충분
