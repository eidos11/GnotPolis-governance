## Structure

```
GnotPolis (Nation)
├── GnoTech (Executive)     — SSOT management, execution (Claude Code)
├── Mnemo_Fantasy (Senate)  — Semantic verification, knowledge hub (Notion)
├── Consilium (Advisors)    — Expert consultation (Web LLMs, CLI tools)
└── Mnemo (Sovereign)       — Final authority
```

## About

GnotPolis is a governance framework for managing a personal knowledge system (GnoTech) that structures cognitive science, business strategy, and logical writing into software-style modular packages.

This repository contains the public-facing governance documents. The knowledge system itself is maintained in a private repository.
