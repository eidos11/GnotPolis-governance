## Decision Gate (4/28) — 외부 증거 수집 현황

| 증거 유형 | 목표 | 현재 | 상태 |
|----------|:----:|:----:|:----:|
| 유상 의향 (구매/결제 의사 표현) | 2건+ | 0 | ⚪ |
| 반복 가능한 문의 | 흐름 확인 | 0 | ⚪ |
| 후기/추천 | 1건+ | 0 | ⚪ |
| 질 높은 대화 (DM/상담/구체 질문) | 추적 | 0 | ⚪ |

**판정 기준**:
- ✅ **계속** = 유상 의향 2건+ OR 반복 문의 흐름 OR 후기 1건+
- 🟡 **이중 트랙** = 무료 반응 있으나 유상 전환 없음
- 🔴 **생존 모드** = 외부 반응 거의 없음
